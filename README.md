# Discord Bot  

## Overview  
This is a simple yet functional Discord bot designed to add some fun and utility to your Discord server. The bot includes commands for generating random numbers, rolling dice, and fetching inspirational quotes.  

## Features  
- 🔢 **/random [min] [max]** – Generates a random number between the given minimum and maximum values. 
- 🎲 **/dice** – Rolls a six-sided dice and returns a random face.  
- 📝 **/quotes** – Fetches and displays a random motivational quote.
- 🌍 **/translate [text] [language]** - Translates a text into selected language.

## Installation  

1. **Clone the Repository**:  
   ```bash
   git clone https://github.com/thatnepalidev/discord-bot.git
   cd discord-bot

2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   
3. **Enter the Discord Token**:
   ```bash
   BOT_TOKEN = "paste your token here"
   
4. **Run the Bot**:
   ```bash
   python bot.py

## Usage
Once the bot is running and added to your Discord server, use the following slash commands:

- `/random [min] [max]` – Get a random number within the specified range.
- `/dice` – Rolls a dice and returns a number between 1-6.
- `/quotes` – Returns a random motivational quote.
- `/translate [text] [language]` - Translates a text into a different language.
